# Atelier

Luxury fashion atelier landing page (Ukrainian) — static site built from the Emergent export with custom updates.

## Preview locally

```bash
npm run preview
```

Then open http://127.0.0.1:8765/

Or:

```bash
python3 -m http.server 8765
```

## Main files

- `index.html` — entry page
- `bundle.js` — React app and styles
- `hero-atelier.svg` — hero illustration
- `maps.html` — map fallback
